数据结构

​	Object

​	使用花括号{}包含的键值对结构，Key必须是string，value为任何基本类型或数据结构

​	Array

​	使用中括号[]来起始，并用逗号，来分割元素

基本类型

string、number、true、false、null

```json
{
    "name":"",
    "age":"",
    "birthdat":"1990-01-01",
    "school":"蓝翔",
    "major":["理发","挖掘机"],
    "has_girlfriend":false,
    "car":null,
    "house":null,
    "comment":"这里是注释"
}
```
